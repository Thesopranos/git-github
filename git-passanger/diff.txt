burada iki commit arasındaki farkları görmeyi öğreneceğiz kıyaslamayı öğreneceğiz
sadece commitleri değil branch'leri de kıyaslayabileceğiz daha da fazlası var
git add . ile git add . yapmadan önceki halini bile kıyaslayabileceğiz

git diff HEAD = son commit ile şu anki dosya arasındaki farkları gösterir

git diff "commit id" "commit id" = iki commit arasındaki değişiklikleri gösterir

git diff "branch name" "branch name" = iki branch arasındaki farkları gösterir
